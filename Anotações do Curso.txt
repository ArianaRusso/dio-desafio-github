SHA (Secure Hash Algorithm): encriptação que gera um conjunto de caracteres identificador.

Objetos do Git: BLOBS, TREES e COMMITS 

Ciclo de vida:
*Untracked (arquivos que o git não tem conhecimento)
*Tracked: 
-unmodified (arquivo ainda não modificado)
-modified (arquivo modificado)
-staged (area de espera)

Ambientes de desenvolvimento: 
-working directory (arquivos untracked)
-staging area (arquivos tracked)
-local repository (composto por commits)

Servidor: remote repository (GitHub)

Conflito de Merge (diferença de versões)
